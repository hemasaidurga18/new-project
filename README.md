# Hostel Room Booking Frontend (React + Vite + Plain CSS)

### Setup Instructions

1. Extract this ZIP and open in VS Code.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the dev server.
4. Open the local URL (default: http://localhost:5173).

### Features
- React with routing (Home, Rooms, My Bookings)
- Plain CSS styling
- LocalStorage for mock booking persistence
- Node.js + Vite environment (frontend only)

### Structure
See `src/` folder for organized components, pages, and styles.
